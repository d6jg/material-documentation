Introduction


Material Skin is a HTML5 WebApp for Mobile and Desktop devices to be used in conjunction with Logitech Media Server 7.9.x


Material is built and copyright by Craig Drummond (cpd73) using Vuetify


Documentation by Jim Gooch (d6jg) using HelpNDoc


Notes

This documentation will frequently refer to http://lmsipaddress:9000/Material - substitute your LMS IP address and Port accordingly





Last modified - 02/11/2019 13:33 - based upon Material 1.1.2



Recent Release History


1.2.0 Release - Imminent

A new feature for 1.2 is the ability to set a password (n Material's server settings) to restrict access to alarms, extra player settings, and server settings. If a password is set on the server, a password field will appear in Material's UI settings. If both of these match, the other features are enabled.


1.1.2 Release - 02/11/2019

Bug fix release. Main changes:

  1. Fix mobile bottom nav layout with Firefox.
  2. Supply album sort when using add-all/play-all - so albums are added in the order shown. Requires LMS from 1st Nov - not sure if its in the current nightly build or not.
  3. Don't show composer if it's the same as artist.
  4. Allow starting a new search from search results list.
  5. Fix HTML rendering for biographies, reviews, etc. when called from 'More' menu.
  6. Fix swipe to change now-playing info tabs on mobile layout.
  7. Fix references to classic skin modification CSS files in releases. This should make the settings more Material-like.


1.1.1 Release -26/10/2019

Bug fix release. Changes:

  1. Fix menu icon missing from grid items.
  2. If keyboard shortcuts are disabled, dont prevent the event from propagating.
  3. Use Alt+Arrow instead of just Arrow for shortcuts.
  4. When remapping icons, check object icon/icon-id property is a string.
  5. Fix broken expand now-playing button in desktop layout.
  6. Fix updating queue when remove a selection of tracks.
  7. Place 'Server settings' back into main menu.
  8. Fix missing shortcuts from some queue action tooltips.
  9. Mark icon buttons as non-selectable to try and prevent copy dialog on mobile.
  10. Tweaks to classic skin used for settings, etc.
  11. Set focus each time a dialog is shown.
  12. Customise shortcut text (Ctrl/Command) depending upon whether OS is Mac or not.
