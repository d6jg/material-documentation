Introduction


Material skin is an HTML5 WebApp for Mobile and Desktop devices to be used in conjunction with Logitech Media Server 7.9.x


Material skin is built and copyright by Craig Drummond (cpd73) using Vuetify


Documentation by Jim Gooch (d6jg) using HelpNDoc



Notes

This documentation will frequently refer to http://lmsipaddress:9000/Material - substitute your LMS IP address and Port accordingly
